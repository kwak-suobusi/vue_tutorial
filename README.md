## Vue.js 튜토리얼

#### 1. 터미널(명령 프롬포트)를 실행하고 cd 명령어로 현재의 디렉토리로 이동 또는 VS코드 상단에 터미널 탭에서 실행하면 해당 폴더에서 터미널이 실행됩니다. 
```
cd 폴더주소
```

####  2. Git 구준엽 말고 클론 후 node_modules를 받기 위해 아래 코드 터미널에서 실행하기.
```
npm install
```

#### 3. 브라우저에서 화면을 보기 위해서는 아래 코드 터미널에서 실행하기.
```
npm run serve
```
